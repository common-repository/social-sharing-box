Add a floating social media box in your posts and make sharing easy. Default services are delicious,facebook,linkedin,reddit,twitter,google+1 and stumbleupon.

== Description ==

Simply download and install the plug-in. No special setup requirements!

NOTE: After you activate the plugin you will need to go to Settings -> Social Box and select the social media services you want to display to each post.

The Social Sharing Box plugin comes with the following services,

*Google +1
*Facebook
*Twitter
*Delicious
*Linkedin
*Reddit
*Stumbleupon

== Installation ==

Follow these steps to install the Social Sharing Box plugin:

1. Download social-sharing-box.zip to your computer.
2. Extract the archive and Upload the `social-sharing-box` folder to your `/wp-content/plugins/` directory.
3. Activate the Social Sharing Box plugin through the 'Plugins' menu in WordPress.
4. Go to Settings -> Social Sharing Box to configure the plugin.
